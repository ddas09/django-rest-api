Supportted Endpoints

  • /api/stocks/

  • /api/stocks/<id>/

Supportted HTTP Methods

Method Endpoint Paramters Action
GET /api/stocks/ None Get all stocks
POST /api/stocks/ Symbol, Name, Price Insert a new stock
GET /api/stocks/<id>/ None Get a particular stock by id
PUT /api/stocks/<id>/ Symbol, Name, Price Update a stock by id
DELETE /api/stocks/<id>/ None Delete a particular stock by id

Examples

  1. GET /api/stocks/

  2. GET /api/stocks/3/

  3. POST /api/stocks/

    Request Body: {'symbol': 'FB', 'name': 'Facebook Inc'}

  4. PUT /api/stocks/2/

    Request Body: {'symbol': 'TWTR', 'name': 'Twitter Inc'}

  5. DELETE /api/stocks/3/
